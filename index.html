Credit Card Checker

Valid credit card numbers have a particular pattern to them. Luhn's algorithm
is used to verify this pattern. More information about the algorithm can be found here.
If the card number does not match the pattern it can be determined that the credit
card number is not valid. This tool performs the algorithm and tells you whether the
card number entered is valid or not.
